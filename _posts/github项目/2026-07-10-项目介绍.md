## 项目介绍

本系列记录安全工具与攻防项目的学习与实践，覆盖从静态分析到动态防护的完整技术栈。

### IAST · 交互式应用安全测试

**技术栈**：Java Agent、ASM 字节码插桩

**核心工作**：实现 Http / Source / Propagator / Sink 四类插桩点，设计 TaintContext + TaintTag 污点追踪模型（ThreadLocal + IdentityHashMap），跑通完整数据流链路。

**成果**：形成来源检测 vs 内容检测的对比认知，理解 IAST 与 RASP 本质差异。

### RASP · 运行时应用自我保护

**技术栈**：ByteBuddy → ASM、Java Agent、agentmain

**核心工作**：三阶段递进——Visitor 模式基础插桩 → 12 条指令检测逻辑注入（+ Bootstrap ClassLoader 可见性解决）→ 动态 Attach + HTTP 端点热开关。

**成果**：从 Demo 升级为可投产 Agent，覆盖 Runtime.exec 六个重载。

### SAST · 静态应用安全测试

**技术栈**：Tai-e（南大软件分析）、Java

**核心工作**：数据流分析（活跃变量 + 常量传播 + 死代码检测 + 过程间常量传播）→ 指针分析（2-obj 上下文选择器 + 别名感知）→ 污点分析（Source/Sink/TaintTransfer 插件化架构）。

**成果**：构建数据流 → 指针 → 污点的 Y 字形完整学习路径。

### DAST · 黑盒漏洞扫描平台

**技术栈**：Go、Vue、Asynq、nmap、nuclei

**核心工作**：剖析 VulnScope 五阶段扫描流水线（域名解析 → 存活探测 → 端口扫描 → 指纹识别 → 漏洞扫描），理解异步任务调度、四级队列权重、QPS 限速与生命周期管理。

**成果**：掌握生产级 DAST 平台的调度与架构设计。

### Nautilus · C2 框架免杀实践

**技术栈**：C/C++、Windows API、直接 Syscall

**核心工作**：编译期——多层字符串加密 + API 动态解析 + Ntdll Unhooking；运行时——回调执行 + 反沙箱 + 间接 Syscall + 调用栈欺骗 ROP gadget + Ekko Sleep 数据页加密；后处理——Section 重命名 + 熵值降低 + 导入表稀释 + Authenticode 签名。

**成果**：VT 检出从 16/71 优化至 7/71，形成编译期/运行时/后处理三层纵深免杀体系。
